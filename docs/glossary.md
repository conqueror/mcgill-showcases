# Glossary

## ATE
Average treatment effect across all units.

## CATE
Average treatment effect within a subgroup.

## Counterfactual
What would have happened under a different action for the same unit.

## Uplift
Incremental outcome caused by treatment.

## Semi-Supervised Learning
Learning from a small labeled set plus a large unlabeled set.

## Self-Supervised Learning
Learning representations from data without manual labels.

## Active Learning
Selecting which unlabeled samples to label next for maximal gain.
