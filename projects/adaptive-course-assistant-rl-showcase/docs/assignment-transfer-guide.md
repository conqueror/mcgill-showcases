# Assignment Transfer Guide

If you want to reuse the *idea* of this showcase in your own work, change the right things.

## Good Transfer

- pick a different domain,
- define your own state variables,
- define your own actions,
- explain your own reward tradeoffs,
- write your own evaluation criteria.

## Weak Transfer

- keep the tutoring story but rename a few variables,
- keep the same reward and pretend it is new,
- copy the artifact set and only change the headings.

Use this sentence as a check:

> if I removed the project name from my assignment, would the structure still obviously look copied from the showcase?

If the answer is yes, you have not transferred enough.
