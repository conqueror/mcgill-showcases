# Build and Extend This Project: Skill Playbook

This file is for contributors who want to evolve the project itself.

## Recommended Skill Order

1. `core-qna-synthesis` (deep)
- Use for clarifying goals, scope, and learning outcomes before writing code.

2. `core-writing-plans`
- Use for implementation planning in small, reviewable steps.

3. `eng-python-engineer`
- Use for robust Python implementation and refactoring.

4. `core-test-driven-development`
- Use to add tests before feature changes.

5. `eng-smart-test-runner`
- Use for efficient and relevant test execution.

6. `mermaid-diagrams`
- Use for architecture and concept visuals in documentation.

7. `core-oracle-second-opinion` (optional)
- Use for an extra review pass on design and explanation quality.
