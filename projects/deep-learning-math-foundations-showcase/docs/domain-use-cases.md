# Domain Use Cases

## Domain Use Cases

### Retail

- Vectors can represent demand features for a product-week pair.
- Probability summaries describe uncertainty in demand or stockout events.
- Gradient descent intuition transfers directly to model training.

### Healthcare

- Matrix transformations can represent feature projections or embedding changes.
- Derivatives describe sensitivity of outcomes to predictors.
- Cross-entropy shows up in many classification tasks.

### Finance

- Probability estimates can represent default likelihoods or fraud events.
- Entropy can help explain uncertainty in classification outputs.
- Optimization traces build intuition for parameter updates in risk models.

### Education

- This project is useful as a pre-neural-network bridge because it explains the math in runnable, inspectable form.
