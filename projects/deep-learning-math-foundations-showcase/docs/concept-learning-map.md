# Concept Learning Map

## Concept Map

| Concept | Where to inspect | What to notice |
|---|---|---|
| Vector addition and scaling | `artifacts/vector_operations.csv` | Operations become numeric building blocks for model computation |
| Matrix transformations | `artifacts/matrix_transformations.csv` | Rotations and scaling change vector representations systematically |
| Derivatives | `artifacts/derivative_examples.csv` | Derivatives measure how sensitive outputs are to input changes |
| Probability summaries | `artifacts/probability_simulations.csv` | Mean, variance, and estimated probability summarize uncertainty |
| Entropy and cross-entropy | `artifacts/information_theory_summary.md` | Uncertainty and prediction mismatch are quantifiable |
| Gradient descent | `artifacts/gradient_descent_trace.csv` | Iterative updates reduce loss step by step |

## How To Use This Map

- Start from an artifact.
- Explain its meaning in plain language.
- Open the matching module in `src/deep_learning_math_foundations_showcase/`.
- Change one input and rerun the showcase.
