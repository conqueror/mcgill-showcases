# How to Use This Showcase for Assignment 2

Do not copy the showcase. Transfer the design pattern.

1. Pick a new domain.
2. Define your own state.
3. Define your own actions.
4. Define your own reward.
5. Create one simple baseline.
6. Train or simulate one learning policy.
7. Compare behavior, not just reward.
8. Identify one reward hacking path.
9. Write a deploy-shadow-reject memo.

## Domain Delta Statement

- My domain differs because...
- My state differs because...
- My actions differ because...
- My reward differs because...
- My evaluation differs because...
